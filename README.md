# StreamDesk

Version: v13.3

Stream control system for LoCo50-Dev.

## Setup

1. Install dependencies: `npm install`
2. Run server: `npm start`
3. Open control panel: http://localhost:3000/control.html

## Folder Structure

```
├── server.js
├── package.json
├── README.md
└── public/
    ├── control.html
    ├── modules/
    ├── assets/
    ├── fonts/
    ├── user/
    ├── music/
    └── pop/
```
